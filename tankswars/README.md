# Tanks Wars

## Текущая версия

`1.1.0.2v5`

Правило версий: если базовая версия не меняется, следующая правка получает следующий суффикс: `1.1.0.2v2`, затем `1.1.0.2v3` и так далее.

После каждой правки нужно писать, какие файлы были изменены. Все дальнейшие изменения кода, стилей, данных и документации нужно заносить в этот README отдельным блоком версии.

## Запуск

Проект работает как статическая браузерная игра. Запускать лучше через локальный HTTP-сервер, потому что `data.csv` загружается через `fetch`.

```powershell
python -m http.server 8000
```

После запуска открыть:

```text
http://localhost:8000/
```

## Структура

- `index.html` - базовая HTML-структура и подключение файлов.
- `css/styles.css` - все стили интерфейса.
- `data.csv` - данные танков, экономики, снарядов и доступности.
- `img/` - изображения интерфейса и танков.
- `js/00-state.js` - глобальное состояние, режимы боя, настройки, константы.
- `js/01-data.js` - загрузка CSV, сохранения, cookies/localStorage, экипаж.
- `js/02-hangar-tech-tree.js` - ангар, дерево техники, сравнение танков, экипаж.
- `js/03-events-store-profile.js` - события, ежедневные награды и задания, магазин, профиль.
- `js/04-ui-developer.js` - режимы боя, настройки, overlay, developer mode.
- `js/05-battle-maps.js` - карты боя.
- `js/06-battle-logic.js` - логика боя, награды, выживание, статистика.
- `js/07-battle-render.js` - отрисовка боя и HUD.
- `js/08-startup.js` - старт игры, запуск/остановка боя, управление.

## Изменения 1.1.0.2v5

Исправлено:

- Новая кнопка `Назад` теперь показывается не только для экранов, открытых через боковое меню, но и для экранов `Сравнить` и `Экипаж`.
- Причина бага: `Сравнить` и `Экипаж` открывали overlay напрямую и не проходили через общий `openOverlay()`, где включалась кнопка.

Файлы, изменённые в версии `1.1.0.2v5`:

- `README.md`
- `js/02-hangar-tech-tree.js`

## Изменения 1.1.0.2v4

Исправлено:

- Кнопка `Назад` полностью переделана с нуля.
- Старая кнопка `#backButton` внутри `#screenOverlay` скрыта и больше не используется.
- Новая кнопка `.overlayBackButton` создаётся через JavaScript, добавляется напрямую в `#game`, показывается при открытии overlay и скрывается при закрытии.
- Закрытие overlay теперь выполняется на `pointerdown` собственной кнопки, без зависимости от слоёв `overlayContent` и без ожидания события `click`.

Файлы, изменённые в версии `1.1.0.2v4`:

- `README.md`
- `css/styles.css`
- `js/04-ui-developer.js`

## Изменения 1.1.0.2v3

Исправлено:

- Кнопка `Назад` поднята над всеми overlay-слоями через высокий `z-index` и явный `pointer-events: auto`.
- Закрытие overlay теперь ловится на уровне `document` в capture-фазе для `pointerdown`, `pointerup`, `mousedown`, `mouseup`, `click` и `touchstart`.
- Это исправляет ситуацию, когда обычный обработчик кнопки не срабатывал из-за перехвата события другим слоем или обработчиком.

Файлы, изменённые в версии `1.1.0.2v3`:

- `README.md`
- `css/styles.css`
- `js/04-ui-developer.js`

## Изменения 1.1.0.2v2

Исправлено:

- Кнопка `Назад` в overlay теперь принудительно останавливает всплытие события и закрывает текущий экран.
- Глобальная блокировка `mousedown` больше не применяется ко всему документу. Она работает только во время боя и только на `battleCanvas`.

- В сравнении танков список выбора расширен с танков в ангаре до всех загруженных не будущих танков.

## Изменения 1.1.0.2v1

Добавлено и исправлено:

- Ежедневная награда теперь может выдавать технику из контейнерного пула.
- Если ежедневная награда выдала танк, который уже есть у игрока, начисляется компенсация золотом.
- Ежедневные задания уже поддерживают условия по конкретному танку, нации и классу техники.
- Сравнение танков расширено: добавлены нация, класс, все снаряды, урон, пробитие, тип орудия, магазин, разброс, шанс пробития, стоимость и флаги доступности.
- В сравнении зелёным подсвечивается лучшая характеристика, красным худшая. Для задержек и разброса меньшее значение считается лучше.
- Экипаж больше нельзя бесплатно мгновенно прокачать кнопкой. Он качается боевым опытом экипажа, который равен 50% обычного опыта.
- Улучшение члена экипажа за золото оставляет бонус `+5%` к характеристикам этого члена.
- Убраны ссылки на отсутствующие изображения `img/kont.png` и `img/rezimi/*.png`.
