# Tanks Wars Online Local

## Запуск

```powershell
cd C:\Users\ira77\Desktop\landing\tankswars\online
node server.js
```

По умолчанию сервер открывается на:

```text
http://localhost:8021/
```

Для проверки с нескольких браузеров откройте этот адрес в каждом браузере. Для другого устройства в одной сети используйте IP компьютера:

```text
http://<ip-компьютера>:8021/
```

## Что уже работает

- локальный HTTP-сервер без внешних зависимостей;
- WebSocket `/ws`;
- серверная очередь боев;
- матч `7 на 7`;
- несколько браузеров видят один сервер и попадают в серверную очередь;
- сервер принимает snapshot профиля игрока;
- локальный экран аукциона и сохранение ставок в профиле.

## Ограничение текущей базы

Сервер уже собирает игроков в матч, но сама физика боя пока считается в браузере. Следующий этап - перенос боевой симуляции на сервер и рассылка authoritative state всем клиентам.
