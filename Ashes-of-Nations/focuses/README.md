# CSV-фокусы Ashes Of Nations

Файл ветки называется так:

```text
Страна[год].csv
```

Пример:

```text
Россия[2026].csv
```

Основные столбцы:

- `id` - стабильный идентификатор фокуса для сохранений.
- `название` - название карточки.
- `описание` - текст карточки.
- `подробно` - длинное описание для развёрнутой карточки по двойному клику.
- `дни` - длительность фокуса.
- `доступно с` - дата открытия фокуса в формате `ГГГГ-ММ-ДД` или `ДД.ММ.ГГГГ`. Можно оставить пустым.
- `ответвление` - название ветки. Нужно для группировки и блокировок.
- `x`, `y` - ручная позиция в дереве. Можно оставить пустыми.
- `после1` ... `после10` - фокусы, которые идут после текущего.
- `требование1` ... `требование10` - фокусы, которые нужны текущему.
- `награда1` ... `награда10` - награды в формате `ключ=число`.
- `блокировать1` ... `блокировать3` - ответвления, которые будут заблокированы после завершения этого фокуса.
- `случайно1` ... `случайно10` - один из этих фокусов будет автоматически засчитан случайным образом после завершения текущего.

Поддерживаемые ключи наград:

```text
politicalPower, commandPower, stability, warSupport, manpower, factories,
gdp, budget, oil, steel, food, rare, nuclearProgram, nuclearReactors,
nuclearWarheads
```

Также награды могут менять политический режим и страну:

```text
ideology=communist|monarchist|socialist|democratic|national|neutral
reform=central-planning
law:economy=war
law:conscription=extensive
law:trade=closed
rename=Новое название страны
color=#b71924
formedNation=rsfsr
integrateCountry=Белоруссия
```

Для требований и столбцов `после` можно писать либо `id`, либо точное название фокуса.
